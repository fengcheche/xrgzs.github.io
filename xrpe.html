潇然PE启动 Xiaoran PE

潇然PE启动 （Xiaoran PE）是一个从自用角度出发，以联网、集成为定位，围绕 潇然一键装机助理 相关功能定制的 WinPE 维护系统；

本人对 Linux 部署有需求，还增加了 iPXE 网启环境，支持从网络启动最新的 Xiaoran PE、Live-CDs (Clonezilla、GParted、Debian Installer 等)

至于PE是否纯净，主要是看安装工具，我们使用的是潇然一键装机助理、CGI、Dism++、WinNTSetup，手动安装镜像都是纯净的，请放心使用！

如需多功能PE，我不是PE发烧友，无法为您提供，如有需求请前往：USM、Edgeless、FirPE Project、AngelPE、HotPE、CowPE、ZnPE、Glass PE、DawnPE、ComPE等了解其它项目，或前往无忧启动论坛-PE作品发布区查找，如H3多启多模集成PE系统、KuerPE、LightningPE

下载获取

WIM格式启动文件：pe64.wim | pe32.wim

ISO格式合盘（推荐）：XRPE.iso

潇然PE启动制作器（不推荐）：潇然PE启动制作器V1.04(Build 2023.4.24).exe

iPXE 命令：

#!iPXE
dhcp
chain http://url.xrgzs.top/ipxe

使用方法

第一种：使用潇然PE启动制作器制作（不推荐）

第二种：使用Ventoy，直接放入pe64.wim和pe32.wim（大神推荐）（仅 Legacy Boot 推荐，UEFI Boot 会出现分辨率 640x480 的问题）

第三种：使用Ventoy，放入XRPE.iso（推荐使用此方法过渡）

第四种：使用UltraISO、Rufus等工具，将XRPE.iso写入到U盘或刻录到光盘（小白推荐）

第五种：使用主控量产工具，将XRPE.iso写入到U盘的 CD-ROM 分区（卖盘的推荐，会导致U盘报废，请勿随意尝试）

第六种：虚拟机加载XRPE.iso（虚拟机推荐）

第七种：在局域网内一台计算机上运行 TinyPXE网启服务器.exe，其它计算机通过 PXE 网卡启动

第八种：想办法进入 iPXE，在 iPXE 启动界面按住 Ctrl + B，输入上面的 iPXE 命令

本人自用，推荐使用

  • 升级了 ISO 启动文件，修复了 UEFI 启动分辨率不正确的问题

  • 集成了 iPXE 网络启动功能，支持 Legacy Boot 下从 IPv6 / HTTPS 启动潇然PE最新版、安装最新 Linux 发行版

  • 自带 TinyPXE 网络启动服务器，支持 代理 DHCP，仅需双击根目录下的 TinyPXE网启服务器.exe 即可

  • 支持自动连接原系统下的 Wi-Fi，或读取各盘符下潇然一键装机助理备份的 Wi-Fi 密码 xml 配置文件；

  • 使用 PENetwork 加载网络，连接网络 SMB 共享更方便，托盘显示 IPv6 和 DNS 信息，中文 Wi-Fi 名称会显示乱码；

  • 调整默认颜色模式为浅色、DWM 标题颜色为默认颜色（蓝色），微调StartisBack，更换太阳谷图标（来自 pc521）、鼠标光标（来自 FirPE），增加初始化 PE 界面（PECMD LOGO）、 X 盘 WinPE 专属图标；

  • 支持 PXE 自动任务，能够自动向 TFTP 服务器查询下载脚本，无需 iPXE（TFTP.exe因为暂时没需求被砍掉）

提示：启动至少需要 1.3GB 内存，适合 4GB 及以上机器使用！
